1. 项目简介

DragonCoin（DRC） 是一个部署在 BNB Smart Chain（BEP-20） 上的社区驱动型数字价值网络。 项目希望通过公开透明的官网、可验证的合约地址、活跃的社交媒体和清晰的白皮书路径，逐步构建一个让用户可以安心参与的链上生态。

DragonCoin 的核心理念是：“以社区为起点、以透明为基础、以长期主义为方向”。这意味着任何重要变更（网站、社交、白皮书、合约说明）都会通过公开页面对外披露，最大限度减少信息不对称。

2. 生态结构（Ecosystem Overview）

DragonCoin 生态预期由四个层级组成：

  • ① 代币层：以 DRC 代币为生态价值载体，用于表示参与、激励与潜在应用。
  • ② 社区层：由持币者、贡献者、内容创作者、合作伙伴构成，是项目真正的“发动机”。
  • ③ 创意层 / NFT 层：未来可挂载官方视觉资产、纪念徽章、社区贡献凭证，配合 OpenSea 官方账号统一管理。
  • ④ 透明层：通过官网、白皮书、BscScan、X（Twitter）等公开渠道同步进展，让外部审核方也能轻松验证项目。

注：当前版本为“展示型白皮书”，主要目标是证明项目的真实性与可持续性，为后续 Tokenomics 和治理模块预留空间。

4. 愿景与价值主张

DragonCoin 的目标并不仅仅是发行一个代币，更重要的是在社区之间建立一种“可持续的数字信任”。 我们认为，一个好的链上项目需要同时具备：

  • 透明性：官网、社交、白皮书、邮箱可验证，更新有记录；
  • 可用性：能在 BNB 链生态中被轻松识别与集成；
  • 社区性：项目的发展方向不是少数人决定，而是逐步向持币者、参与者开放；
  • 品牌性：有明确的视觉识别（Logo、Banner、NFT），有利于在社交和交易平台中被识别。

因此，当前阶段优先完成的是 “身份验证” + “资料完善”，包括： 官网上线 → About / Whitepaper 页面 → X 账号 → BscScan 信息更新 → OpenSea 官方账号。

5. 路线图（Roadmap · 预览）

  • Phase 1（已完成）：域名与官网上线、品牌视觉确定、合约部署、X 账号建立
  • Phase 2（进行中）：BscScan 项目信息补充与重新提交审核
  • Phase 3（即将开始）：白皮书正式版撰写（含 Tokenomics、治理、生态合作）
  • Phase 4（2025-2026）：社区共建、NFT 纪念系列、外部伙伴接入、工具化支持

路线图会根据社区反馈与市场环境进行微调，最新版本以官网公布为准。

6. 社区与治理

DragonCoin 认为，一个健康的 Web3 项目必须具备可参与性。后续将逐步开放以下社区功能：

  • 社区提案（Proposal）收集通道
  • 代币持有者投票（按持仓或参与度）
  • 社区内容贡献奖励（文章、视频、空间、分发）
  • 官方 NFT 纪念章、限量视觉资产分发

在治理机制上线前，所有更新以官网与官方 X 账号为最终渠道。
