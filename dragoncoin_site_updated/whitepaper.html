1. Introduction

DragonCoin (DRC) is a community-driven digital value network on the BNB Smart Chain (BEP-20). This public preview whitepaper is intended to provide verifiable information for community members and for 3rd-party reviewers (such as BscScan).

The final PDF whitepaper will include tokenomics, governance, ecosystem plans, and technical details. This page is the public entry point.

2. Ecosystem Overview

  • Token Layer: DRC as the core unit of value and participation.
  • Community Layer: holders, contributors, partners.
  • Creative / NFT Layer: future visual assets, badges, collectibles, to be published via official OpenSea.
  • Transparency Layer: website, socials, on-chain data — always public.

This version focuses on making the project verifiable. Utility will be added step by step.
